¿Porque Afip SDK es gratis?

Todo web service deberia contar con su propio SDK ya que programarlo desde cero es una tarea ardua y tediosa que se repetiria cada vez que un desarrollador quiera usar el web service, por eso es que programamos este completo y sencillo SDK para que tu puedas ponerte manos a la obra rapidamente.

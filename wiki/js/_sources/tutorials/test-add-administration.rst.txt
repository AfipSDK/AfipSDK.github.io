Habilitar administrador de certificados de testing
====================================================

Para agregar la aplicación "WSASS - Autogestión Certificados Homologación" a nuestro escritorio de AFIP primero debemos ingresar en "Administrador de Relaciones de Clave Fiscal"

.. figure:: ../_static/images/test-add-administration/1.png
    :target: ../_static/images/test-add-administration/1.png

En caso de tener permiso para administrar las relaciones de otros contribuyentes nos hará elegir el de quien queremos administrar, caso contrario se elegirá el nuestro automáticamente. Siempre en modo testing recomendamos usar en usuario de uno mismo.

.. figure:: ../_static/images/test-add-administration/2.png
    :target: ../_static/images/test-add-administration/2.png


Luego elegimos "Adherir servicio"

.. figure:: ../_static/images/test-add-administration/3.png
    :target: ../_static/images/test-add-administration/3.png

Allí dentro elegimos "AFIP > Servicios interactivos > WSASS - Autogestión Certificados Homologación"

.. figure:: ../_static/images/test-add-administration/4.png
    :target: ../_static/images/test-add-administration/4.png

Le damos a confirmar y ya tendremos acceso a la aplicación desde nuestro escritorio de Afip

.. figure:: ../_static/images/test-add-administration/5.png
    :target: ../_static/images/test-add-administration/5.png
